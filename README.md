# C_Compiler

A Compiler for C 89 to Arm v9 assembly using dynamic memory and register allocation.

Currently implemented features:

-Arithmetic Expressions

-Logical Expressions

-Bitwise Expressions

-Function Definition and Decleration

-Function Calls

-Structures

-While Loops

-For Loops

-Custom Types
